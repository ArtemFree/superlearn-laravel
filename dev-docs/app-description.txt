Веб-приложение для маркетплейса фриланса

Студент создает заявку на проект, заполняет данные, регистрируется

Исполнитель регистрируется, заполняет данные о себе, и может просматривать заявки, может отликаться на них, общаться со студентом

Исполнитель может предварительно написать студенту о себе, сказать, что может выполнить его заказ.

Студент согласовывает Исполнителя и начинается проект. На проект могут подписаться несколько исполнителей

// сущности
Author - firstName, lastName, age, about, status, completedProjects, failedProjects, rating, isBanned, isVerified,


//страницы
/user/:id - страница студента
/author/:id - страница автора
/project/:id - страница проекта
/signup/user - регистрация студента
/signup/author - регистрация автора
/signin - страница входа
/projects?perPage=0&class=0&sumFrom=0&sumTo=0 - страница проектов
/authors - страница с анкетами авторов

// фичи
Есть Администратор со всеми правами c админкой и возможностью править всё
